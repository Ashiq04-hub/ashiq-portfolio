# Project Requirements

## Project Information

**Project Name:** Ashiq Portfolio Website

**Author:** Ashiq Alsinawi

**Version:** 1.0.0

**Purpose:** Personal portfolio website showcasing skills, projects, education, tools, and professional experience for recruiters, employers, and clients.

---

# Project Goals

The website should:

* Showcase software engineering skills
* Present technical projects professionally
* Demonstrate UI/UX and frontend development capabilities
* Provide an easy way for recruiters to contact the owner
* Achieve excellent performance and SEO scores

---

# Target Audience

* Recruiters
* Hiring Managers
* Software Engineers
* Tech Leads
* Freelance Clients

---

* Sticky navigation bar

* Mobile navigation menu
